# 火線超人 FHIRLineBot - 品牌指南

## 專案名稱

**正式名稱**: 火線超人 FHIRLineBot
**英文名稱**: FHIRLineBot
**英文 Tagline**: A pocket smart-on-FHIR visit tracker for everyone.

## 專案願景

把「排隊等待」變成「有準備地就醫」，讓醫院服務真正走進民眾日常生活。

## 核心定位

| 面向 | 說明 |
|------|------|
| **用戶** | 患者、家屬、照護人員 |
| **場景** | 醫院掛號排隊、回診提醒、家屬陪護 |
| **平台** | LINE 官方帳號（無需下載 App） |
| **價值** | 減少焦慮等待、改進就醫體驗、家庭照護支持 |

## 品牌信息

### 主要訊息

"看診到底排到第幾號了？家人現在看完沒？"

這句話詮釋了火線超人要解決的核心問題：
- 患者在診間外的焦慮和不確定性
- 家屬對親人就醫進度的關切
- 缺乏即時資訊導致的時間浪費

### 解決方案

一個加到 LINE 裡就能用的就醫小幫手：
- 無需下載額外 App
- 綁定醫院就診資料
- 一鍵查詢看診進度
- 門診資訊和回診提醒

### 核心承諾

讓患者和家屬能夠：
- 即時掌握看診進度
- 減少在診間外的焦慮等待
- 更有準備地應對就醫過程
- 改善整體就醫體驗

## 功能架構

### 核心功能

1. **看診進度查詢**
   - 實時查看隊伍位置
   - 預估等待時間
   - 叫號提醒

2. **診療資訊管理**
   - 預約掛號
   - 病歷查詢
   - 用藥記錄

3. **家屬陪護功能**
   - 親人位置分享
   - 代理查詢
   - 照護提醒

4. **智能提醒**
   - 回診通知
   - 用藥提醒
   - 檢驗提醒
