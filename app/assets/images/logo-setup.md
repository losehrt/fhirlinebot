# FHIRLineBot Logo Setup

## Instructions

1. Save the orange superhero logo image as one of these files:
   - `app/assets/images/fhirlinebot-logo.png`
   - `app/assets/images/fhirlinebot-logo.jpg`
   - `app/assets/images/fhirlinebot-logo.svg`

2. The logo should be the exact image provided:
   - Orange superhero figure with cape
   - Flame symbol on chest
   - "FHIRLineBot" text below
   - Orange color theme (#E97132)

3. The image will be automatically resized in the application to fit the UI components.

## Usage
The logo is used in:
- Sidebar header (36x42px display size)
- Login page
- Other branding locations
