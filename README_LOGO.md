# Logo 設置指南

## 請將 logo 圖片檔案放置到以下位置之一：

1. `app/assets/images/fhirlinebot-logo.png`
2. `app/assets/images/fhirlinebot-logo.jpg`
3. `app/assets/images/fhirlinebot-logo.svg`

## Logo 規格

- 檔名必須是 `fhirlinebot-logo` 加上副檔名
- 圖片會自動縮放到適當大小（高度 42px）
- 建議使用透明背景的 PNG 或 SVG 格式

## 放置步驟

1. 將您提供的橘色超人 logo 圖片下載
2. 重命名為 `fhirlinebot-logo.png` (或 .jpg/.svg)
3. 放置到 `app/assets/images/` 資料夾
4. 重新整理頁面即可看到 logo

如果沒有放置 logo 圖片，系統會顯示 "FHIR" 文字作為替代。
