# FHIR LINE Bot

一個基於 Rails 框架開發的 Smart on FHIR 應用 MVP，透過 LINE Bot 作為民眾與醫院即時溝通的橋樑。

## 專案目的

本專案旨在建立一個整合 FHIR (Fast Healthcare Interoperability Resources) 標準與 LINE Bot 的醫療溝通平台，讓民眾能夠透過熟悉的 LINE 介面，與醫療機構進行即時且安全的互動。

## 主要功能

- 整合 Smart on FHIR 標準，實現醫療資料的互通性
- 透過 LINE Bot 提供便利的使用者介面
- 作為民眾與醫院之間即時溝通的橋樑
- 安全的醫療資訊存取與管理

## 技術架構

- Ruby 3.4.2
- Rails 8.0.4
- PostgreSQL
- Tailwind CSS
- Hotwire (Turbo + Stimulus)
